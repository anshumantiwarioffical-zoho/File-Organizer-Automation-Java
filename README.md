File Organizer Automation Tool (Java)
Project Overview

This is a Java console-based automation tool that organizes files in a given folder into subfolders based on file type.
The program scans a directory, checks each fileâ€™s extension, and moves it into the appropriate folder such as Images, Documents, Videos, or Others.

Key Features:

Automatically sorts files into categories based on type

Supports image files: jpg, png, jpeg

Supports document files: pdf, txt, docx

Supports video files: mp4, avi

Moves all other file types into an "Others" folder

Easy to run and modify for any folder path

Fully console-based, no external frameworks required
